# Phone Country Codes


### Usage
```php
// Return the list of phone codes:
\megastruktur\PhoneCountryCodes::getCodesList();

/* Return the full array of codes and details:
   *  "country_code" => [
   *    "name",
   *    "dial_code",
   *    "code",
   *  ]
*/
\megastruktur\PhoneCountryCodes::getCodesFullList();
```
